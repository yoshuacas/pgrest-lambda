# Task 02: Auth Schema Initialization Module

**Agent:** implementer
**Design:** docs/design/gotrue-native-auth.md

## Objective

Create `src/auth/schema.mjs` — the module that holds the DDL
statements for the `auth` schema and provides
`ensureAuthSchema(pool)` for one-time initialization.

## Target Tests

From Task 01, `src/auth/__tests__/schema.test.mjs`:

- `ensureAuthSchema` executes all DDL statements in order
- `ensureAuthSchema` is a no-op on second call
- `ensureAuthSchema` re-executes after `_resetInitialized`

## Implementation

**New file: `src/auth/schema.mjs`**

Export an array `AUTH_SCHEMA_SQL` containing six DDL statements,
each as a separate string (DSQL single-statement constraint):

1. `CREATE SCHEMA IF NOT EXISTS auth`
2. `CREATE TABLE IF NOT EXISTS auth.users (...)` — columns:
   - `id UUID PRIMARY KEY DEFAULT gen_random_uuid()`
   - `email TEXT NOT NULL`
   - `encrypted_password TEXT NOT NULL`
   - `email_confirmed_at TIMESTAMPTZ DEFAULT now()`
   - `role TEXT NOT NULL DEFAULT 'authenticated'`
   - `aud TEXT NOT NULL DEFAULT 'authenticated'`
   - `app_metadata JSONB NOT NULL DEFAULT '{"provider":"email","providers":["email"]}'::jsonb`
   - `user_metadata JSONB NOT NULL DEFAULT '{}'::jsonb`
   - `created_at TIMESTAMPTZ NOT NULL DEFAULT now()`
   - `updated_at TIMESTAMPTZ NOT NULL DEFAULT now()`
3. `CREATE UNIQUE INDEX IF NOT EXISTS auth_users_email_idx ON auth.users (email)`
4. `CREATE TABLE IF NOT EXISTS auth.refresh_tokens (...)` — columns:
   - `id BIGINT GENERATED BY DEFAULT AS IDENTITY (CACHE 1) PRIMARY KEY`
   - `token TEXT NOT NULL`
   - `user_id UUID NOT NULL`
   - `parent TEXT`
   - `revoked BOOLEAN NOT NULL DEFAULT false`
   - `created_at TIMESTAMPTZ NOT NULL DEFAULT now()`
   - `updated_at TIMESTAMPTZ NOT NULL DEFAULT now()`
5. `CREATE UNIQUE INDEX IF NOT EXISTS auth_refresh_tokens_token_idx ON auth.refresh_tokens (token)`
6. `CREATE INDEX IF NOT EXISTS auth_refresh_tokens_user_id_idx ON auth.refresh_tokens (user_id)`

Export `async function ensureAuthSchema(pool)`:
- Check module-level `initialized` flag. If true, return.
- Loop through `AUTH_SCHEMA_SQL`, calling `await pool.query(sql)` for each.
- Set `initialized = true`.

Export `function _resetInitialized()` — sets `initialized = false`. For tests only.

**Key constraints:**
- No foreign keys (DSQL limitation).
- `BIGINT GENERATED BY DEFAULT AS IDENTITY (CACHE 1)` instead
  of `SERIAL` (DSQL-compatible).
- `gen_random_uuid()` supported by DSQL and PostgreSQL 13+.
- Each DDL is a separate string — no multi-statement batches.

## Test Requirements

No additional unit tests beyond what Task 01 defines. The three
schema tests in `schema.test.mjs` fully cover this module.

## Acceptance Criteria

- `src/auth/schema.mjs` exports `AUTH_SCHEMA_SQL`,
  `ensureAuthSchema`, and `_resetInitialized`.
- All three tests in `src/auth/__tests__/schema.test.mjs` pass.
- Existing tests still pass: `npm test`.
